# Sourcing Assistant

AI-помощник для IT-рекрутеров, который помогает быстро анализировать вакансии и генерировать Boolean-запросы для поиска кандидатов на hh.ru.

## Возможности

* Анализ текста вакансии
* Определение целевой должности
* Генерация Boolean-запроса для hh.ru
* Выделение обязательного стека технологий
* Выделение дополнительных поисковых технологий

## Технологии

* Streamlit
* Google Gemini API
* Python
